Poly-filters

Useful filters for Polymer expressions
Fork Star Issue

Filters:

  • Uppercase
  • Lowercase
  • Trim
  • Round
  • Reverse
  • Replace
  • Length
  • Parsing link
  • Date
  • Array filter

Getting started

1. Install with Bower. 
        bower install poly-filters
2. Import Polymer component inside HEAD tag. 
            <link rel="import"
                href="bower_components/poly-filters/poly-filters.html">
3. PROFIT!!!! 
            <span>{{'Hello world!' | uppercase }}</span>

Uppercase

{{ value | uppercase }}

Lowercase

{{ value | lowercase }}

Trim

{{ value | trim }}

Round

{{ value | round }}

Reverse

{{ value | reverse }}

Replace

{{ value | replace('World', 'Folks') }}

Length

{{ value | length }}

Parse link

{{ value | link('pathname') }}

Date

{{ value | date('yy-MM-dd HH:mm t') }}
